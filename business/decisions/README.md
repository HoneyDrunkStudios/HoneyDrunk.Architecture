## Conventions

- Filename: `BDR-NNNN-short-slug.md`
- Status values: **Proposed** · **Accepted** · **Superseded** · **Rejected**
- Sections (recommended): Context · Options · Decision · Consequences · Action Items
- When a BDR is superseded, leave the old file in place and link forward
