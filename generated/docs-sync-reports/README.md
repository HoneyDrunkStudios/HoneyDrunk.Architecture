# Docs Sync Reports

This directory stores one report per `docs-sync` run at `generated/docs-sync-reports/{YYYY-MM-DD}.md`.

Each report summarizes repos scanned, skipped repos with reasons, report-only findings, and any docs-sync PRs opened or updated. Reports must be safe to summarize into Discord through the ADR-0086 runner notification path.
