# Generated ADR Drafts

This directory contains ADR drafts generated from architecture discussions.

Files here are **in-progress** — they move to `/adrs/` once accepted.

## Naming Convention

`ADR-DRAFT-{kebab-case-title}.md`

## Workflow

1. Agent facilitates architecture discussion
2. Draft is generated here
3. Human reviews and refines
4. Once accepted, moved to `/adrs/ADR-{number}-{title}.md`
5. Draft is deleted
