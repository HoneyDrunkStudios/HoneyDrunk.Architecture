# Generated Issue Packets

This directory contains generated GitHub Issue packets ready for handoff.

Files here are **ephemeral** — they can be deleted after the issues are created in target repos.

## Naming Convention

`{YYYY-MM-DD}-{repo-short-name}-{kebab-case-description}.md`

## Workflow

1. Agent generates packet from a discussion or planning session
2. Human reviews the packet
3. Issue is created in the target repo (manually or via automation)
4. Packet is deleted or archived
